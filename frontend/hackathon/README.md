# 🚀 SkillQuest AI

SkillQuest AI is a gamified learning platform that helps users learn HTML, CSS, and JavaScript through interactive quizzes.

## Features

- 🔐 Login System
- 🌍 Mission Selection
- 📝 HTML, CSS & JavaScript Quizzes
- ⭐ XP, Coins & Level System
- 🪐 Planet Evolution
- 🛍️ Planet Shop
- 🤖 AI Explanation for Answers
- 📊 Progress Bar
- 🏆 Achievement System
- 💾 Local Storage Support

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage
